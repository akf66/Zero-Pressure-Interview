# 1. 生成共享Model库 
```shell
cd server/shared
kitex -module haha ./../../idl/rpc/user.thrift

## 其中haha 替换为自己的go mod 包名称
```

# 2. 创建 Kitex Rpc 服务
```shell
cd server/cmd/user
kitex -service user \
      -module haha \
      -use haha/server/shared/kitex_gen \
      ./../../idl/rpc/user.thrift
      
## 其中haha 替换为自己的go mod 包名称
```

# 3. 创建 Hertz Http 服务
```shell
cd server/cmd/api
hz new -idl ./../../idl/http/user.thrift -module haha/server/cmd/api

```
